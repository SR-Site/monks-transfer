# Contributing to the Frontend Skeleton

In short:

- create tickets for bugs/features/enhancemants, see [Contributing Tickets](https://mediamonks.assembla.com/spaces/mediamonks-front-end-tools/wiki/Skeleton_-_Tickets)
- write code in a branch (from master) and create a merge request for review, see [Contributing Git](https://mediamonks.assembla.com/spaces/mediamonks-front-end-tools/wiki/Skeleton_-_Git)
 
Read more about contributing on the [Contributing Wiki page](https://mediamonks.assembla.com/spaces/mediamonks-front-end-tools/wiki/Contributing_to_the_Skeleton_Project)
